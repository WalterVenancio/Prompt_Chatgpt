# Central de Prompts

Esta pasta agora possui uma pagina inicial chamada `index.html` que reune os 3 geradores:

- `Prompt_Proposta_Comercial_AAC.html`
- `Gerador de Composição de Custos.html`
- `Gerador de Descrições — Planilha Orçamentária.html`

## Como publicar no GitHub

1. Crie um repositorio no GitHub.
2. Envie todos os arquivos desta pasta para o repositorio.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch principal e a pasta `/ (root)`.
6. Salve.

Depois disso, o GitHub Pages vai abrir o arquivo `index.html` automaticamente como pagina inicial.

## Como usar

1. Abra o link do GitHub Pages.
2. Escolha um dos 3 geradores na lateral.
3. Preencha os campos.
4. Use o botao de copiar dentro do gerador.

## Observacao

Os tres arquivos originais continuam separados, mas agora ficam acessiveis por uma unica pagina central.
